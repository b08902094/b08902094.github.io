Movie Recommendation System (Multithreading)

The program uses a movie dataset (60000+ movies) to filter out movies using keywords inputted by the user. Each movie has a hot vector that identifies if it belongs to a certain genre. The program then sort the movies in an order that matches the user's preference to the ones that are least likely to match. Multithreading merge sort is used to speed up the execution time by dividing the filtered movies into several parts and assigning each part to a thread.
